
= = = BREADTEXT = = =

A text editor by Jack Eisenmann, for Jack Eisenmann.

NOTE: This text editor is still being tested for bugs.

FEATURES

> You can edit text
> Copy, cut, and paste from system clipboard
> Undo and redo
> Find and replace
> Customizable indentation
> Set and goto marks
> Macro playback
> Compatible with macOS and Linux

INSTALLATION

To install:

make
sudo mv ./breadtext /usr/local/bin/breadtext

On Linux you must install xclip for clipboard functionality:

sudo apt-get install xclip

CONTROLS

For help with controls, enter "/help" after running BreadText.
